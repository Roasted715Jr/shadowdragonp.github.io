Welcome to Zawada

Zawada is spinoff of the popular rage-inducing, cheater-promoting, and braggart-creating game. Originally by Andrew McGovern, Zawada was created by ShadowDragon Studios. Same rules, same logic, different game!

The rules are simple.

  1. Each page shows you a riddle presented in paragraphs and/or pictures.
  2. Think of the answer to the riddle and type the url below in the web address bar:

    https://shadowdragonp.github.io/portals/games/zawada/youranswer.html

  3. Replace youranswer with the answer you had in mind. This should be self-explanatory, but statistics show that 80% of the American population is "mentally challenged."
  4. If your answer is correct, the next level will appear. If not, you will be given a 404 error.
  5. The answer is usually never shown right off the bat. This game is meant to test your intelligence, so sometimes you will have to think outside of the box.

  6. When you're ready, begin!
